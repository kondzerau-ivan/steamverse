# SteamVerse

**SteamVerse** – это социальная сеть для геймеров, которая использует возможности Steam API для аутентификации через Steam OpenID и получения профиля пользователя. Проект разделён на две части:

- **Сервер (Backend):** на базе [Express.js](https://expressjs.com/). Сервер реализует API‑эндпоинты, обеспечивает аутентификацию (с помощью Passport и стратегии [passport‑steam](https://www.npmjs.com/package/passport-steam)) и управляет сессиями через [express-session](https://www.npmjs.com/package/express-session).

- **Клиент (Frontend):** одностраничное приложение (SPA) на React, где для маршрутизации и управления состоянием используются библиотеки из экосистемы [TanStack](https://tanstack.com/). Это позволяет эффективно кэшировать и обновлять данные, а также организовать клиентскую часть в виде модульного приложения.

## Основная концепция

**1. Аутентификация через Steam**  
Используя Steam OpenID, пользователи смогут быстро регистрироваться и входить в систему без необходимости создавать отдельный профиль. После успешного входа система автоматически получит данные Steam-профиля, такие как имя, аватар, список игр, достижения и другая информация. Это позволит создать единый, качественный профиль, отражающий игровую историю пользователя.

**2. Профили пользователей**  
После аутентификации каждый профиль можно расширить дополнительными функциями:  
- **Основная информация Steam** – отображение имени, аватара, стима-идентификатора и базовой статистики.  
- **Игровая статистика** – автоматическое обновление информации о количестве игр, времени проведённом в играх, достижениях.  
- **Лента активности** – публикация обновлений, связанных с достижениями, новым прохождением или добавлением в список игр, полученной непосредственно из Steam API.  

**3. Социальные функции**  
Сеть будет включать в себя знакомство и взаимодействие с другими игроками:  
- **Система друзей и подписок** – используя данные Steam, можно рекомендовать пользователям знакомых или игроков с похожими интересами, установить уведомления о новых достижениях друзей.  
- **Форумы и чаты** – создание групповых обсуждений, обмен сообщениями, организация тематических сообществ по играм или жанрам.  
- **События и турниры** – возможность создавать и присоединяться к мероприятиям, матчмейкингу для кооперативного игрового процесса.

### Дополнительные возможности

**Геймификация платформы:**  
- Внедрение системы достижений и рейтингов не только из Steam, но и собственных, стимулирующих участие в жизни сети.  
- Возможность сравнения статистики и организации рейтинговых таблиц по разным играм или достижениям.

**Контент и монетизация:**  
- Разрешение пользователям публиковать обзоры, советы и гайды по играм, делиться скриншотами и видео.  
- Внедрение механизмов рекомендаций, например, подбирающих события или турниры на основе профиля и игровой статистики.

**Интеграция с сообществами Steam:**  
- Возможность импортировать данные о группах и обсуждениях из Steam, что позволит пользователям легко переходить между платформами и расширять свою игровую сеть.

### Роадмап реализации

1. **Прототипирование:**  
   - Разработать базовую систему аутентификации через Steam OpenID.  
   - Создать минимальный UI для регистрации и отображения данных профиля.

2. **Интеграция данных Steam:**  
   - Настроить регулярное обновление профилей пользователей с использованием Steam API.  
   - Реализовать ленту активности, основанную на данных, полученных из Steam.

3. **Разработка социальных функций:**  
   - Внедрить систему друзей, чат и создание групп.  
   - Добавить функции публикаций, комментариев и лайков.

4. **Тестирование и масштабирование:**  
   - Провести нагрузочное тестирование, обеспечить безопасность данных.  
   - Подготовить систему к возможной монетизации и интеграции дополнительных функций.

### Заключение

Проект **SteamVerse** представляет собой обширное решение для геймеров, которое объединяет в себе функционал социальной сети с особенностями, характерными для экосистемы Steam. Такой подход не только сокращает время на регистрацию, но и позволяет использовать богатые данные, предоставляемые Steam, для создания персонализированного опыта. Если проект будет дорабатываться, можно добавить системы рекомендаций, анализ игровых предпочтений и даже элементы внутриигровой экономики, что сделает платформу уникальной на рынке.

Если интересуют более детальные технические детали, примеры кода для аутентификации или идеи по геймификации платформы, можем обсудить это дальше.
